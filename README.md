* Sistema de Análise Preditiva

* *  Sobre o projeto
Este projeto está sendo desenvolvido em Python com o objetivo de analisar estatísticas de partidas de futebol e futuramente gerar previsões baseadas em dados historicos. A ideia é evoluir o sistema até que ele seja capaz de comparar as duas equipes e calcular probabilidades utilizando diferentes indicadores estatísticos.

* * Tecnologias utilizadas

- Python
- Request
- API-Footbaal

* * Funcionalidades implimentadas 

- busca de equipes pela API
- Seleção da equipe desejada
- Consulta das partidas
- Cálculo de vitórias, empates e derrotas
- Cálculo de gols marcados e sofridos
- Cálculo de média de gols
- Cálculo de aproveitamento
